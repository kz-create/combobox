## アプリケーション起動方法

### 前提

- node、pnpm がインストールされている
- `docker compose`コマンドが実行できる

- 下記コマンドでサーバを起動後、[localhost:5173](localhost:5173)にアクセス

```sh
pnpm install
pnpm dev
```

## Swagger UI の閲覧方法

- プロジェクトルートで下記コマンドを実行し、コンテナを起動後、[localhost:8000](localhost:8000)にアクセス

```sh
docker compose up -d
```
